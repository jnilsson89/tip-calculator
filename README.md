# Tip Calculator

## Overview
This project is an Xamarin.Android application that automatically calculates the tip for restaurant bills. It is made following a tutorial from Micosoft Learn (see link below).

## Languages
* C#
* XML

## Frameworks
* .NET
* Xamarin

## IDE
